PROJECT TITLE: Ship Harbor Simulation
PURPOSE OF PROJECT: To simulate different ships in a harbor and the time it takes them to unload given certain conditions
VERSION or DATE: 10/24/2014
HOW TO START THIS PROJECT: Run Harbor.java or right click on Harbor and select main()
AUTHORS: Tena Percy and Haley Daniels
USER INSTRUCTIONS: Once the program has started you will be prompted for the number of ships, enter the number
of desired ships to simulate and press enter. If any results are 0 it is because they are very small decimals.
You will be prompted to run the simulation multiple times, if you wish to exit return any key other than 'Y'
